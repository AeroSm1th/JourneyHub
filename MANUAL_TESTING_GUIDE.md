# JourneyHub 手动测试指南

本文档详细描述了 JourneyHub 目前已完成的所有功能（Phase 1 + Phase 2），并提供逐步手动测试步骤。

## 前置准备

### 环境要求
- Node.js >= 18
- 已配置 `.env` 文件（包含 `VITE_SUPABASE_URL` 和 `VITE_SUPABASE_ANON_KEY`）
- Supabase 项目已创建并执行了建表脚本
- Supabase Storage 已创建 `avatars` bucket

### 启动应用
```bash
npm install
npm run dev
```
浏览器访问 `http://localhost:5173`

## 已完成功能总览

| 模块 | 功能 | 路由 |
|------|------|------|
| 首页 | 落地页展示 | `/` |
| 认证 | 注册、登录、退出 | `/auth/login`, `/auth/register` |
| 地图 | 交互式地图、城市标记、愿望清单标记 | `/app/map` |
| 城市管理 | 创建、查看、编辑、删除城市记录 | `/app/cities`, `/app/cities/:id` |
| 愿望清单 | 添加、查看、删除、转换为城市记录 | `/app/map`（侧边栏切换） |
| 行程规划 | 创建、编辑、删除行程，日程管理，待办事项 | `/app/trips`, `/app/trips/new`, `/app/trips/:id` |
| 统计可视化 | 热力图、柱状图、饼图、排行榜 | `/app/insights` |
| 个人资料 | 编辑昵称、上传头像、导出数据、删除账户 | `/app/profile` |
